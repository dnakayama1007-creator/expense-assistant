# 📱 LINE Bot セットアップガイド

## 経費精算アシスタント × LINE 連携

LINEでメッセージ（テキスト・領収書写真）を送るだけで、
経費アシスタントに自動登録される仕組みを作ります。

---

## 🔧 手順1: LINE公式アカウントを作成する

### 1-1. LINE Official Account Manager にアクセス

👉 **https://account.line.biz/signup** にアクセス

- 「LINEアカウントでログイン」を選択
- 普段使っている LINE アカウントでログインします

### 1-2. 公式アカウント情報を入力

| 項目 | 入力内容 |
|------|---------|
| アカウント名 | `経費アシスタント`（お好きな名前でOK） |
| メールアドレス | あなたのメールアドレス |
| 業種 | 「個人」→「個人（その他）」でOK |

「確認する」→「完了する」で公式アカウントが作成されます。

---

## 🔧 手順2: Messaging API を有効にする

### 2-1. LINE Official Account Manager で設定

1. https://manager.line.biz/ にログイン
2. 作成したアカウントを選択
3. 右上の **「設定」** をクリック
4. 左メニューから **「Messaging API」** を選択
5. **「Messaging APIを利用する」** ボタンをクリック

### 2-2. プロバイダーの設定

- 「プロバイダーを作成」を選択
- プロバイダー名: `自分の名前` や `My Apps` など（何でもOK）
- プライバシーポリシー・利用規約は空欄でOK
- 「OK」を押して完了

---

## 🔑 手順3: チャネルアクセストークンとシークレットを取得

### 3-1. LINE Developers コンソールにアクセス

👉 **https://developers.line.biz/console/** にアクセス

- LINE アカウントでログイン
- 手順2で作成したプロバイダーをクリック
- 作成されたチャネル（Messaging API）をクリック

### 3-2. チャネルシークレットをコピー

1. **「チャネル基本設定」** タブを開く
2. ページ下部にある **「チャネルシークレット」** をコピー
   - 📋 `abc123def456...` のような文字列です

### 3-3. チャネルアクセストークンを発行

1. **「Messaging API設定」** タブを開く
2. ページ下部にある **「チャネルアクセストークン（長期）」** セクション
3. **「発行」** ボタンをクリック
4. 表示されたトークンをコピー
   - 📋 長い文字列（100文字以上）です

---

## ⚙️ 手順4: Webhook設定

LINE Bot サーバーを起動した後に設定します。

### LINE Developers コンソールで：

1. **「Messaging API設定」** タブを開く
2. **Webhook URL** に以下を入力:
   ```
   https://あなたのURL/webhook
   ```
   ※ ミニPCのサーバーのURLを入力します（後でご案内します）
3. **「Webhookの利用」** をオンにする

### 応答設定（Important!）：

1. **LINE Official Account Manager** → 設定 → 応答設定
2. **応答メッセージ** → **オフ** にする
3. **あいさつメッセージ** → お好みでオン/オフ

---

## 📝 取得した情報のメモ

以下の2つを控えてください:

```
チャネルシークレット: ________________________________

チャネルアクセストークン: ________________________________
```

この2つをサーバーの設定ファイルに入力すれば、LINE Bot が動きます！

## ❓ よくある質問

**Q: 料金はかかりますか？**
A: 無料です。月200通（2025年6月以降）のメッセージ送信が無料枠に含まれています。
   受信（Webhook）は無制限で無料です。

**Q: 他の人に見られますか？**
A: いいえ。あなたとBotの1対1のやり取りです。

**Q: スマホのLINEアプリから使えますか？**
A: はい！作成したBotを友だち追加すれば、普通のLINEトークと同じように使えます。
